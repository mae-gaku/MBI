# Setup

1. git clone
```
git clone https://github.com/mae-gaku/MBI.git
cd MBI
```

2. install
```
npx shadcn@latest init
npx shadcn@latest add input
npx shadcn@latest add sidebar
```


3. run

**FastAPI**
```
uvicorn main:app --reload
```

**Next.js**
```
npm run dev
```

4. アクセス
```
http://localhost:3000/
```
